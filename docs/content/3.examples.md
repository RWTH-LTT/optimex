# Examples

This section contains example applications of `optimex` demonstrating various use cases and features.

::alert{type="info"}
**All examples are available as interactive Jupyter notebooks!** Check out the [Examples & Notebooks](/notebooks) section for detailed, executable notebooks that you can run yourself.
::

## Available Examples

The examples are available as Jupyter notebooks that you can run interactively:

- **[Basic optimex Example](/notebooks/basic_optimex_example)** - Introduction to the core concepts and basic usage
- **[Two Decision Layers Example](/notebooks/basic_optimex_example_two_decision_layers)** - Advanced example with multiple decision layers
- **[Hydrogen Production Pathway](/notebooks/h2)** - A case study on hydrogen production pathway optimization
- **[Carbon Dioxide Removal](/notebooks/cdr)** - CDR pathway optimization example
- **[Methanol Production](/notebooks/methanol)** - Methanol production pathway optimization

## Running Examples Locally

You can download and run these examples on your own machine:

1. Clone the repository:
```bash
git clone https://github.com/RWTH-LTT/optimex.git
cd optimex
```

2. Install dependencies:
```bash
conda create -n optimex -c conda-forge -c cmutel -c diepers optimex jupyterlab
conda activate optimex
```

3. Launch Jupyter:
```bash
jupyter lab notebooks/
```
