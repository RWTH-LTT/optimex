# Examples

This section contains example applications of `optimex` demonstrating various use cases and features.

## Available Examples

The examples are available as Jupyter notebooks that you can run interactively:

- **Basic optimex Example** - Introduction to the core concepts and basic usage
- **Mini Hydrogen Case** - A case study on hydrogen production pathway optimization

## Running Examples Locally

You can download and run these examples on your own machine:

1. Clone the repository:
```bash
git clone https://github.com/RWTH-LTT/optimex.git
cd optimex
```

2. The example notebooks are located in the `notebooks/` directory.

3. Launch Jupyter:
```bash
jupyter lab
```
